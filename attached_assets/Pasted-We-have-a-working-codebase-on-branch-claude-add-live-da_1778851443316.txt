STEP 5 — Confirm the two processes use separate ports

The API server and the Vite web server cannot share the same PORT. In your workflow configuration:

API server must start with PORT=<api_port> (e.g. 3001)
Web server must start with PORT=<web_port> API_PORT=<api_port> BASE_PATH=/ (e.g. PORT=8080, API_PORT=3001)
The Vite config already has the /api proxy configured — it reads API_PORT to know where to forward requests. This is the root cause of all APIs failing: without the proxy, every /api/* call hits Vite (which has no routes) and 404s.
